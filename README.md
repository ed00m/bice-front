# BICE-FRONT

Tecnologías ocupadas

- Angular 9.1.11
- Sass

Metologías ocupadas

- BEM (Block, Element, Modifier)

## Pasos a seguir

Levantar primero proyecto **bice-api**

Instalar dependencias que contiene la aplicación

`$ npm install`

Ejecución de aplicación

`$ ng serve`

URL de la aplicación

`http://localhost:4200`

Ejecución pruebas unitarias

`ng test`
